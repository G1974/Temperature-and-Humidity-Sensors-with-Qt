A modern Weather Station Dashboard built with Qt / QML & C++, designed to display real-time temperature and humidity data from sensors via Serial Port (Arduino/ESP32), combined with live weather forecast data from the Open-Meteo API.  
Features
Indoor temperature monitoring
 Humidity monitoring
Live outdoor temperature & weather icons
Serial communication using QSerialPort
REST API integration with QNetworkAccessManager
Responsive UI built with Qt Quick / QML
Automatic weather updates every 30 minutes
Technologies
C++
Qt 6
QML / Qt Quick Controls
Open-Meteo API
Serial Communication

Perfect for Raspberry Pi projects, Arduino-based systems, smart home dashboards, and embedded weather monitoring applications.
